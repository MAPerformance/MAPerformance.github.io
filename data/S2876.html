Mystère de la Vendition de Joseph

Lieu: Péronne
Région: Somme/Picardie
Date: 1550
Informations clefs:
  • Récurrence // Sujet : Ecritures saintes //
  • Pentecôte //
  • Habitué // Clerc-acteur //
  • Place publique // Publiquement //
Bibliographie: Petit de Julleville, 1880, t. 2. p. 123. – Grenier, 1856. p. 407. // Biblioth. nation, mss. Collection de Picardie, t. CL VIII, fol. 83 //

Dix-sept années plus tard, trois clercs habitués de Saint-Fursy sollicitèrent l'autorisation de jouer en public la Vendition de Joseph aux fêtes de la Pentecôte. Les idées du chapitre s'étaient modifiées. La permission fut refusée « pour plusieurs raisons qu'on exposa de vive voix. »
